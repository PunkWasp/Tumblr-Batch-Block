Tumblr Batch Block

Tumblr Batch Block is no longer being developed by RealityFabric.

If you would like to continue development, please fork this project. If you have questions please email me.

Instructions:

  1. Navigate to the 'Edit appearance' page for your blog.
  2. Click the button with the action you would like to take.

Please note: Both blocking and exporting may take some time, but you can continue to surf the web in another tab while the app runs. When exporting, the list will appear at the top of the page in the sidebar on the right.
